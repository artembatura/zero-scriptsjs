# @zero-scripts/config.webpack

## Options

| Option    | Type      | Default |
| --------- | --------- | ------- |
| sourceMap | _Boolean_ | `true`  |

## Example

`package.json`

```json
{
  "devDependencies": {
    /* preset, which uses config.webpack */
  },
  "@zero-scripts/config.webpack": {
    "sourceMap": false
  }
}
```
